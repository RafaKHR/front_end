Frameworks Java Script

Framework é um termo que se refere a estratégias e ações que visam solucionar um tipo de problema. Contudo, a sua definição vai muito além do mercado de TI ou de software. Sendo assim, essa é uma forma encontrada pelas empresas de otimizar os seus resultados através de abordagens prontas.

JavaScript ou JS, é uma linguagem de programação que permite a você implementar itens complexos em páginas web - toda vez que uma página da web faz mais do que simplesmente mostrar a você informação estática - mostando conteúdo que se atualiza em um intervalo de tempo, mapas interativos ou gráfico 2D/3D animados, etc. Logo, Frameworks de Java Script são frameworks voltados para o uso de js, normalmente utilizado por profissionais para criar seus próprios designs de páginas web.
